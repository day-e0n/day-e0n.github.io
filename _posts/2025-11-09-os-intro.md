## 프로세스와 스레드

운영체제의 가장 기본적인 개념인 프로세스와 스레드에 대해 알아봅시다.

### 프로세스 (Process)

프로세스는 **실행 중인 프로그램**을 의미합니다.

```c
#include <stdio.h>
#include <unistd.h>

int main() {
    pid_t pid = fork();
    
    if (pid == 0) {
        printf("자식 프로세스: PID = %d\n", getpid());
    } else {
        printf("부모 프로세스: PID = %d\n", getpid());
    }
    
    return 0;
}
```

### 스레드 (Thread)

스레드는 프로세스 내에서 실행되는 **작업의 단위**입니다.

- 같은 프로세스의 스레드들은 메모리를 공유합니다
- 컨텍스트 스위칭 오버헤드가 프로세스보다 적습니다

### 차이점

| 구분 | 프로세스 | 스레드 |
|------|---------|--------|
| 메모리 | 독립적 | 공유 |
| 생성 비용 | 높음 | 낮음 |
| 통신 | IPC 필요 | 직접 접근 |
